# Chrome Remote Desktop on Compute Engine

## Overview
Deploy **Chrome Remote Desktop** on **Compute Engine** VMs for remote access to Windows/Linux desktops.

## Use Cases
*   Remote access to full desktop environment.
*   Graphical applications (CAD, video editing, etc.).
*   Supports both Windows and Linux VMs.

## Components
*   **Compute Engine VM**: Windows or Linux with desktop environment.
*   **Chrome Remote Desktop**: Installed on VM for remote access.
*   **Authentication**: Google account.

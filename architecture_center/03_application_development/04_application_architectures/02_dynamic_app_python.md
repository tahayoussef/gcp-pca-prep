# Dynamic Web App with Python and JavaScript (Jump Start Solution)

## Overview
Deploy-ready dynamic web application using **Python** (backend) and **JavaScript** (frontend) on **Cloud Run**.

## Components
*   **Cloud Run**: Hosts containerized Python backend.
*   **Cloud SQL**: Database.
*   **Cloud Storage**: Static assets.

## Benefits
*   Serverless, fully managed.
*   Easy customization for different use cases.

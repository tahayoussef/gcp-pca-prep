# Three-Tier Web App (Jump Start Solution)

## Overview
Classic three-tier architecture (presentation, business logic, data) deployed on **Cloud Run** (serverless).

## Tiers
*   **Presentation**: Frontend (Cloud Run).
*   **Business Logic**: Backend API (Cloud Run).
*   **Data**: Cloud SQL database.

## Benefits
*   Serverless deployment of traditional three-tier pattern.
*   Automatic scaling and high availability.

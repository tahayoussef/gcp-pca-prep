# Serverless E-commerce Platform (Jump Start Solution)

## Overview
Ready-to-deploy e-commerce application ("Avocano") running on **Cloud Run** (serverless).

## Architecture
*   **Frontend**: Cloud Run service (React/Next.js).
*   **Backend**: Cloud Run service (API).
*   **Database**: Cloud SQL (PostgreSQL).
*   **Autoscaling**: Cloud Run automatically scales based on traffic.

## Features
*   **Serverless**: No infrastructure management.
*   **Observability**: Cloud Trace for distributed tracing.
*   **Cost-Effective**: Pay only for requests.

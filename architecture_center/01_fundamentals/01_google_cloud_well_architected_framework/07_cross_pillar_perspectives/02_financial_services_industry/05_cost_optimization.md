# FSI perspective: Cost optimization

- **Visibility & Governance**
    - **FinOps**: Centralize monitoring using **Cloud Billing** reports and **FinOps Hub**.
    - **Waste Reduction**: Use **Active Assist** to automatically identify idle resources (VMs, DBs) and committed use discounts.

- **Cost Allocation**
    - **Accountability**: Aim for **100% cost allocation** to business units.
    - **Tagging**: Implement a robust tagging strategy (e.g., `team=banking`, `env=prod`) to track spend.
    - **Hierarchy**: Use the **Project/Folder hierarchy** to align cloud structure with the organizational budget structure for automatic allocation.

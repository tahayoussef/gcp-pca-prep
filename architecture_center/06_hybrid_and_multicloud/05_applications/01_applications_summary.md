# Hybrid/Multicloud Applications

## Hybrid Render Farm
Media rendering distributed across on-prem and cloud. **Use**: Burst to cloud for peak rendering loads; cost-effective for VFX studios.

## Connecting Other CSPs with GCP
Patterns for connecting Google Cloud to AWS, Azure, Oracle Cloud:
*   **VPN**: Simple, encrypted, lower bandwidth.
*   **Dedicated Interconnect**: High bandwidth, low latency, via colocation.
*   **Partner Interconnect**: Via connectivity partners (e.g., Equinix, Megaport).

## Use Cases
*   Multi-cloud applications needing cross-cloud data access.
*   Migration from other clouds to Google Cloud.
*   Best-of-breed services from multiple clouds.
